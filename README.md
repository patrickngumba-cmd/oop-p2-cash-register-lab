# OOP Cash Register

This project uses Object-Oriented Programming in Python to simulate an e-commerce cash register.

## Features
* **Add Items:** Adds items to a transaction, calculating total price based on quantity.
* **Discounts:** Applies a validated percentage-based discount to the total.
* **Void Transactions:** Removes the last item purchased and correctly adjusts the total and item list.
